📋 ARCHIVOS CREADOS:
───────────────────────────────────────────────────────────────
✅ app.py                    - Aplicación principal Streamlit
✅ environment.yml           - Configuración entorno Conda
✅ iniciar_app.sh           - Script de inicio automático
✅ README.md                - Documentación completa
✅ ACTXPROG_filtrado.csv    - Catálogo de actividades filtradas

🚀 CÓMO INICIAR LA APLICACIÓN:
───────────────────────────────────────────────────────────────

Opción 1 (Más fácil):
$ ./iniciar_app.sh

Opción 2 (Manual):
$ streamlit run app.py

📌 La aplicación se abrirá automáticamente en:
   http://localhost:8501

🔍 FUNCIONALIDADES:
───────────────────────────────────────────────────────────────

1. BÚSQUEDA POR PACIENTE:
   - Ingresa el número de documento
   - Ve el historial completo de atenciones
   - Descarga los resultados en CSV

2. BÚSQUEDA POR ACTIVIDAD:
   - Selecciona una actividad del menú
   - Ve todos los pacientes que la han recibido
   - Descarga los resultados en CSV

📊 DATOS QUE SE MUESTRAN:
───────────────────────────────────────────────────────────────

Por Paciente:
- ID interno, tipo doc, sexo, nombre
- Historial de atenciones con fechas
- Código y descripción de actividades
- Estadísticas de atenciones

Por Actividad:
- Lista de pacientes
- Documentos y datos personales
- Fechas de atención
- Estadísticas agregadas

💡 CONSEJOS:
───────────────────────────────────────────────────────────────

- La primera carga puede tardar unos segundos
- Los datos se cachean para mejor rendimiento
- Puedes exportar cualquier resultado a CSV
- La interfaz es responsive y fácil de usar

❓ SOLUCIÓN DE PROBLEMAS:
───────────────────────────────────────────────────────────────

Si la aplicación no inicia:
1. Verifica que todos los archivos CSV estén presentes
2. Asegúrate de tener las dependencias instaladas:
   $ pip install streamlit pandas numpy

Si hay errores de memoria:
- El sistema está optimizado para archivos grandes
- Solo carga columnas necesarias de CAB_FAC.csv
